# VEL Mobile

Projeto acadêmico desenvolvido em equipe durante o curso Full-Stack Java do Instituto Proa em parceria com o Senac (2024), com apresentação final em Demo Day.

O VEL é uma plataforma web voltada à gestão logística de entregas, criada para resolver um problema real identificado em pesquisa de campo: a maioria dos entregadores registrava pedidos manualmente, o que gerava erros de pagamento e atritos entre entregadores e empresários. A plataforma digitaliza esse processo, oferecendo controle de comandas, rotas e pagamentos de forma transparente para donos de restaurantes e empresas de entrega.
